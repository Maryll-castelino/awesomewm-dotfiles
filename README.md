### To use my dotfiles:
1. cd into your .config folder
```
cd .config
```
2. backup your existing configuration
```
mv awesome myawesome-backup
```
3. clone this repository
```
git clone https://github.com/Maryll-castelino/dotfiles.git awesome
```
4. restart awesome
